Authorized testing helper

Mail.tm Attacker / Victim Test Accounts

Static pages for creating disposable Mail.tm accounts, checking inboxes, reading messages, and extracting verification links.

Use only for systems where you have permission to test. Powered by Mail.tm.

Attacker Open attacker inbox Create and monitor the attacker test email. Victim Open victim inbox Create and monitor the victim test email.

Quick usage

  1. Open Attacker page and click Load domains.
  2. Click Create email.
  3. Repeat on Victim page.
  4. Use those two emails for registration, invite, reset-password, or email-verification testing.
  5. Click Refresh inbox or enable auto refresh.
